# 停車場指紋辨識系統 （指紋解碼 and 密碼鎖 ）
透過密碼鎖或是指紋辨識打開停車場的閘門，並回傳相關資訊到 手機的 app 上。

功能介紹:
1. 使用數字鍵盤解碼
2. 指紋解鎖
3. 新增指紋
4. 瀏覽解鎖記錄
5. 刪除指紋
6. 更改密碼
7. 閘門紀錄回傳到使用者的 app
